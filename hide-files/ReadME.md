## Summary
- this tool will hide or unhide the files chosen by the user depending on the selection (hide/unhide).
- the hidden files will not be visible in the file explorer even if the file explorer is set to to 'show hidden files'
***
#### Dependencies
- none.
#### Date
- 2025.11.20 as YYYY.MM.DD
#### Python version
- Python 3 (tested on 3.13)
