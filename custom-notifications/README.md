#### **[BlankTools](https://github.com/tromoSM/BlankTools/)** / [custom-notifications](https://github.com/tromoSM/BlankTools/tree/main/custom-notifications)
****
- create custom notifications.
#### Preferences
- all the preferences will be controlled by user inputs
  
| Input title | input types | Description | Example | additional info |
|-|-|-|-|-|
| Enter an app title | str | App title | BlankTools  | Leaving blank may cause [WinError -2147024809] | 
| Enter an title | str | main title | BlankTools notification |  | 
| Enter an description | str | description | blanktools notification description |  | 
| Enter an path for your icon | str | image path | ./icons/img.png | `svg` is not supported. | 
| Choose an audio choice | int | audio choice | 6  |  | 
| Looping audio after playback | bool/(y/n) | audio looping preference | true | not case sensetive | 
#### Dependencies
- All the dependecies can be found in [requirements.txt](Requirements.txt)
